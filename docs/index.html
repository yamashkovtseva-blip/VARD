FROM events
WHERE created_at > now() - interval '90 days'
GROUP BY 1, 2
ORDER BY mau DESC;

Why Vard

Three things every other notebook gets wrong.

Frequently asked

Is Vard really free? +
Yes. BSL 1.1 today, transitioning to Apache 2.0 in 4 years. Free for internal use, forever.
Can I use my own LLM (OpenAI, Claude, local)? +
Yes. Ollama by default; OpenAI-compatible API supported.
What databases do you support? +
Postgres today. DuckDB, Snowflake, BigQuery, Redshift, ClickHouse on the roadmap (open in GitHub Discussions).
Is my data sent anywhere? +
No. Vard is fully self-hosted. Even AI runs locally.
Do you have SOC 2? +
In progress. Target Q3 2026. We help Enterprise customers with their own audits.
How is Vard different from Briefer / Hex / Jupyter? +
See the comparison table above. Short version: only Vard combines self-hosting, local AI, and real-time collab in one product.
